# myrgic-ai

Redirect app for myrgic.ai. Immediately redirects to https://myrgic.com/.
Renders the Myrgic mark and wordmark during the brief redirect window.

## Configuration

See `site.yaml` for the deployment spec.

## Build

`build.sh` copies `src/` into `dist/`, pulls `eigen-form.js` and `colors_and_type.css` from packages/.
